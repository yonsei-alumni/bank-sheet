# 통장 거래내역 변환기
신한은행 모임통장에서는 계좌를 개설한 모임장만이 입금내역을 홈페이지에서 엑셀로 다운로드를 받을 수 있습니다.
이 프로그램은 모임장이 아닌 사람도 입금내역을 단순반복 작업없이 Excel로 변환할 수 있도록 도와줍니다.

이 프로그램은 PIN을 입력하면 내부적으로 신한은행에 로그인하여 보유한 모임통장 목록을 불러오고, 선택한 통장의 거래내역을 엑셀 파일로 변환해 줍니다. (기존에 필요했던 DevTools로 HTML을 복사·저장하는 수동 단계가 제거되었습니다.)

## 준비사항

이 프로그램은 PC에 설치된 Chrome의 로그인 프로필을 사용해 신한은행에 접속합니다.

1. **Chrome** 브라우저가 PC에 설치되어 있어야 합니다.
2. Chrome에서 [신한은행 모바일 페이지](https://m.shinhan.com/mw/pg/SP0102S0200F01?mid=270040220100&groupId=493705)에 접속해 **신한 인증서**를 최초 1회 발급·등록해 두어야 합니다.
    * 신한 인증서가 등록되어 있지 않으면 프로그램 실행 중 안내 대화상자가 나타나며, "Chrome 실행" 버튼으로 등록 페이지로 바로 이동할 수 있습니다.
    * 신한 인증서는 신한은행 모바일앱의 'SOL 패스 인증' 등을 통해 발급받을 수 있습니다.

## 사용법

### 프로그램 다운로드

[Releases](https://github.com/yonsei-alumi/bank-sheet/releases/latest) 페이지에서 운영체제에 맞는 파일을 받으세요.

| 파일 | 운영체제 |
|------|----------|
| `bank-sheet-linux-amd64` | Linux (x86_64) |
| `bank-sheet-windows-amd64.exe` | Windows (x86_64) |

### 엑셀 변환

1. **프로그램 실행**
   다운로드한 실행 파일을 엽니다. 최초 실행 시 내부 브라우저(Playwright Chromium, 약 170MB)가 자동으로 다운로드되며, 수 분 정도 소요될 수 있습니다.

2. **로그인 & 통장 불러오기**
   신한은행 6자리 PIN을 입력하고 "로그인 & 통장 불러오기" 버튼을 누릅니다. 등록된 신한 인증서로 자동 인증되며, 성공하면 보유한 모임통장 목록이 화면에 표시됩니다.

3. **통장 선택**
   내역을 내려받을 모임통장을 하나 이상 선택합니다. "모두 선택 / 해제" 버튼으로 한 번에 토글할 수 있습니다.

4. **조건 설정**
    * **조회 기간** — 1개월 · 3개월 · 6개월 중 선택
    * **정렬** — 과거순 · 최신순
    * **출력**
        * `계좌별 파일` — 선택한 통장마다 별도의 xlsx 파일 생성
        * `하나의 엑셀(시트별)` — 하나의 xlsx 파일 안에 통장별 시트로 생성
    * **저장 위치** — 엑셀 파일이 저장될 폴더. "폴더 변경" 버튼으로 변경할 수 있습니다. (기본값: 프로그램 실행 위치)

5. **엑셀로 다운로드**
   "엑셀로 다운로드" 버튼을 누르면 선택한 통장의 거래내역이 순차적으로 저장되며, 하단 진행 로그와 상태바에 진행 상황이 표시됩니다.

6. **결과 확인**
   완료 후 나타나는 대화상자에서 "예"를 누르면 저장 폴더가 바로 열립니다.

저장 경로는 지정한 저장 위치 아래 `YYYYMMDD/` 하위 폴더에 생성됩니다.

#### 엑셀 파일 형식

| 열 | 내용 | 형식 |
|----|------|------|
| 입금일자 | 거래 일시 | YYYY-MM-DD HH:mm:ss |
| 입금자 | 입금한 사람 이름 | 문자 |
| 입금액 | 입금 금액 | 숫자 (천 단위 콤마) |
| 잔액 | 거래 후 잔액 | 숫자 (천 단위 콤마) |
